‰ΩøÁî®`Function.prototype.apply()`Â∞Ü`this`‰∏ä‰∏ãÊñáÂ∫îÁî®‰∫éÂáΩÊï∞Âπ∂Êèê‰æõÂøÖË¶ÅÁöÑ`ÂèÇÊï∞`„ÄÇ

	return function() {
